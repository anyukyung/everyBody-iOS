## Feature
1. **Camera**   
다양한 포즈에 맞게 신체를 촬영할 수 있습니다. 
2. **Panorama Album**    
전신 / 상체 / 하체로 분류된 사진을 스와이프하며 몸의 변화를 확인할 수 있고, 그리드 모드로 레이아웃을 변경할 수 있습니다.
3. **Video Export**   
부위별로 저장된 사진을 묶어 하나의 동영상으로 만들 수 있습니다. 
4. **Push Notification**   
시간과 요일을 지정하면, 설정된 시간에 맞게 사진을 찍을 수 있도록 푸시 알림을 보냅니다.
